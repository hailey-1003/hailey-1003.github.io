最近在做 Vue+Element-UI 项目呢！

2021-01-25
2021-01-25
杂谈

最近再写一个电商后台管理系统~~

整体采用了前后端分离技术，前端技术是基于 Vue 技术，后台是在网上找了一个搭好的后台（毕竟我也不会写后台嘛emmmm）然后用 Postman 测试了一下接口，都可以正常使用！~ 数据库也是和后台代码一起在网上找的一起的，用 Mysql+Navicat，测试了一下也都没啥问题~

前端技术主要就是：Vue + Vue-router + Element-UI + Axios + Echarts

项目也是直接使用 Vue-cli 脚手架直接创建的。

感觉 Vue + Element-UI 这种开发模式和之前一个个页面写，一行行html和css一起写的开发模式完全不一样！！！

之前在学校写的项目就是使用 jQuery+bootstrap 写的，现在对比来，就感觉自己原来写的都是啥啊哈哈哈哈哈！那样写真的太繁琐啦！所以说框架就是牛逼啊！

但是其实现在也不是特别熟悉这种开发模式，所以还是要在多写几个项目！

但是还是感觉自己还有好多东西没学啊！！还得去看 JS高级程度设计那本书，我还想了解一下 Vue 底层怎么封装的，还得了解一下设计模式，大佬还说至少的能够手撕 Promise，反正就尽力而为吧，学不完我也没办法了emmmm，加油！！！
